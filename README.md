# MERN CRUD Application

This is a full-stack CRUD application built using the MERN stack.

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB

## Features
- Add User
- View Users
- Update User
- Delete User

## Installation

### Clone the repository
git clone https://github.com/NAVEENKUMAR-AGHRAHARAM/mern-crud-project.git

### Install dependencies
cd client
npm install

cd ../server
npm install

### Run project
npm start
